Get Started with Git

What is Git?

Git is the most commonly used version control system. Git tracks the changes you make to files, so you have a record of what has been done, and you can revert to specific versions should you ever need to. Git also makes collaboration easier, allowing changes by multiple people to all be merged into one source. More information can be found on this website:

More about github.

Get started with Git.

Step 1

Create a new "repository" (project) on github.

Step 2

Copy (or clone) the repository to your local machine. Use the commans git clone and paste the github link to your repo. The link to the repo can be found on github.

Step 3

to push changes of your project use the git push origin comamnd.

Step 4

To pull changes from github to your local directory use the gut pull origin command.
