Get Started with Terminal

Terminal Commands

pwd

print working directory- prints absolute path

ls

lists files and directories in current directory

cd

changes the current directory being used in terminal

mkdir

creates a new directory at current location

mv

moves/renames files and directories

touch

creates an empty file

vi

opens files in the vi text editor

git init

initializes current directory for git version control

git add

stages files to be commited

git commit -m

commits changes to git version control with a message

git checkout -b

creates a new branch and changes checks out to it

git push

pushes local git changes to a remote repository/branch

Docker commands

docker ps

lists all running containers

docker compose up

creates and runs a docker container based on docker compose file

docker compose down

shuts down a docker container based on docker compose file

docker kill

kills and shuts down containers based on container id
